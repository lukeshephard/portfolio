## üöÄ Getting Started

1. **Clone the repo**

    ```
    git clone https://github.com/lukeshephard/portfolio.git
    cd portfolio
    ```

2. **Install dependencies**

    ```
    npm install
    ```

3. **Run the development server**

    ```
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) to view it in your browser. Edits will appear live.
